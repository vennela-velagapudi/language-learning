German for Beginners

Start your journey of learning German with the basics of the language, from pronunciation to vocabulary and grammar essentials.

Alphabet & Pronunciation

Special characters: ä, ö, ü, ß
Letters different from English: w (v), v (f), j (y)
Common sounds: ch, sch, ei, ie

Essential Phrases

Hallo = Hello
Tschüss = Bye
Bitte = Please
Danke = Thank you
Guten Morgen = Good morning
Guten Tag = Good day
Guten Abend = Good evening
Wie geht's? = How are you?
Sehr gut = Very good
Sprechen Sie langsam = Speak slowly

Numbers 0-10

0 = Null
1 = Eins
2 = Zwei
3 = Drei
4 = Vier
5 = Fünf
6 = Sechs
7 = Sieben
8 = Acht
9 = Neun
10 = Zehn

Personal Pronouns

ich = I
du = you (informal)
er/sie/es = he/she/it
wir = we
ihr = you (plural informal)
Sie = you (formal)
sie = they

Basic Verbs (Present Tense)

sein = to be

ich bin = I am
du bist = you are
er/sie/es ist = he/she/it is

haben = to have

ich habe = I have
du hast = you have
er/sie/es hat = he/she/it has

Essential Vocabulary

Days:

Montag = Monday
Dienstag = Tuesday
Mittwoch = Wednesday
Donnerstag = Thursday
Freitag = Friday
Samstag = Saturday
Sonntag = Sunday

Colors:

rot = red
blau = blue
grün = green
gelb = yellow
schwarz = black
weiß = white

Articles

Definite(the):

der = the (masculine)
die = the (feminine)
das = the (neuter)
die = the (plural)

Indefinite:

ein = a/an (masculine)
eine = a/an (feminine)
ein = a/an (neuter)

Question Words

Was? = What?
Wer? = Who?
Wo? = Where?
Wann? = When?
Warum? = Why?
Wie? = How?

Basic Adjectives

groß = big/tall
klein = small
neu = new
alt = old
schön = beautiful
gut = good
schlecht = bad

Restaurant Phrases

Die Rechnung, bitte = The bill, please
Ich möchte... = I would like...
Haben Sie...? = Do you have...?
Zum Wohl! = Cheers!
Die Speisekarte, bitte = The menu, please

Shopping Vocabulary

Was kostet das? = How much is that?
Das ist zu teuer = That's too expensive
die Größe = the size
der Preis = the price
die Anprobe = fitting room

Emergency Phrases

Hilfe! = Help!
Ich brauche einen Arzt = I need a doctor
Wo ist das Krankenhaus? = Where is the hospital?
Rufen Sie die Polizei! = Call the police!
